## 📁 Structure du dépôt

Chaque notebook (`.ipynb`) de ce dépôt correspond à un projet indépendant. Le nom du fichier décrit le sujet traité. Chaque notebook contient :
- un contexte et un objectif en introduction
- la démarche d'analyse (exploration, traitement, modélisation selon le cas)
- les visualisations et résultats
- une conclusion avec les limites identifiées

## 🛠️ Stack technique

`Python` · `Pandas` · `Scikit-learn` · `Seaborn / Matplotlib` · `WNTR` · `Power BI`

Les bibliothèques spécifiques utilisées varient selon le projet (voir les imports en tête de chaque notebook).
